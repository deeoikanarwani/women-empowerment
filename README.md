# 🌸 SheRise – Women Empowerment Platform

A full-stack web application designed to promote women empowerment by building a supportive and interactive community. Users can join the movement, share their stories, and engage with others.

---

## 🚀 Features

* 🌍 **Join the Movement**
  Users can register and become part of the community.

* 💬 **Community Stories**
  Users can share their personal experiences and success stories.

* ❤️ **Like System**
  Engage with stories by liking them.

* 📩 **Contact Form**
  Users can send messages or feedback.

* 📊 **Supporters Counter**
  Displays real-time number of people who joined the movement.

* 🧾 **Messages Dashboard**
  View submitted messages.

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/deeoikanarwani/women-empowerment.git
```

2. Navigate to the project folder

```
cd women-project
```

3. Install dependencies

```
npm install
```

4. Run the server

```
node server.js
```

5. Open in browser

```
http://localhost:3000
```

## 📸 How It Works

* Users fill forms (Join / Contact / Stories)
* Data is sent to backend APIs
* Server processes and stores data
* Frontend dynamically displays updated content
